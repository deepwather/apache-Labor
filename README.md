
Apache Labor
============

Dies ist eine Sammlung von Apache Anleitungen, welche von der Compilierung in mehreren
"Labors" zu einer vollen Apache-Konfiguration mit einem Schwergewicht auf der
Sicherheit führen. Das runde Dutzend Tutorials sind aufeinander aufbauend gestaltet
und steigern mit jedem Schritt das KnowHow in den Gebieten Webserver, HTTP SSL/TLS, 
ModSecurity, Logfile Visualisierung.

## Vorbedingungen
Für diese Labors benötigt man eine Linux Installation (vorzugsweise aus der
Debian-Familie, wie bzw. Ubuntu) und minimale Linux Shell Kenntnisse.
Andere Gnu/Linux Derivate sind ebenfalls geeignet; einzig die Paketnamen dürften von
den im Text verwendeten Paketnamen abweichen.


##Inhalt

- [labor-01](labor-01) Kompilieren eines Apache 2.4 Webservers
- [labor-02](labor-02) Konfigurieren eines minimalen Apache Servers
- [labor-03](labor-03) Aufsetzen eines Apache/PHP Applikationsservers
- [labor-04](labor-04) Konfigurieren eines SSL Servers
- [labor-05](labor-05) Das Zugriffslog ausbauen und auswerten
- [labor-06](labor-06) ModSecurity einbinden
- [labor-07](labor-07) OWASP ModSecurity Core Rules einbinden
- [labor-08](labor-08) OWASP ModSecurity Core Rules tunen
- [labor-09](labor-09) Einen Reverse Proxy konfigurieren
- [labor-10](labor-10) Apache effizient konfigurieren und debuggen
- [labor-11](labor-11) Logfiles visualisieren
- [labor-12](labor-12) Verkehr komplett mitschreiben und entschlüsseln
